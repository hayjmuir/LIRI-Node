# LIRI-Node



### This App is built with Javascript and utilizes Node.js

## App Commands
* concert-this "band name here"
* spotify-this-song "song name here"
* movie-this "movie title here"
* do-what-it-says 



## Technologies Utilized 
  * JavaScript
  * Node.Js
  * Moment.Js
  * OMDB API
  * Spotify API
  * Bands in Town API
  * Inqurier
  * Axios
